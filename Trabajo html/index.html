A

Aplicacion transaccional
Aplicación informática orientada al procesamiento de transacciones. Las transacciones son unidades de datos que proceden del exterior de un sistema informático y que desencadena un proceso que se traduce en una actualización de datos permanente y en alguna salida suplementaria.

B

Bit por segundo
Unidad básica de medida de la información en las transmisiones informáticas o las telecomunicaciones. La palabra bit es la abreviatura de BInary digiT. Mientras los números decimales tienen 10 posibles valores (desde 0 a 9), los bits sólo tienen dos posibles valores: 0 o 1. Por consiguiente, un número binario está compuesto sólo por ceros y unos.

Bit
Unidad básica de información representada por ceros y unos que se van sucediendo para conformar los distintos significados.

C

cable coaxial
Cable formado por un conductor central de cobre, recubierto de material aislante, un conductor exterior concéntrico y una cubierta protectora exterior. Suelen emplearse dos tipos de cable coaxial para las redes locales: cable de 50 ohms, para señales digitales, y cable de 75 ohms, para señales analógicas y para señales de alta velocidad.

Canal de datos
Canal de comunicaciones utilizado para la transmisión de datos.

D

Data warehouse
Sistema informático, generalmente instalado en el servidor de una empresa, que permite acceder a un gran volumen de información a partir del cruce de varias bases de datos. Está basado en una colección de datos orientados a un tema, integrados, no volátiles e historiados, organizados para el apoyo de un proceso de ayuda a la decisión. Es una base de datos en la que hay una réplica conjunta de todas las bases de datos operacionales.

Disco duro
Disco, generalmente instalado en el interior del ordenador, en el cual se graba la información mediante tecnología magnética, para recuperarla posteriormente gracias a una o varias cabezas lectoras-grabadoras. Su tamaño se mide en gigabytes (Gb). 1 Gb son 1.024 Mb.

E

Enterprise Java-Beans
Componente hecho en software que se puede reutilizar y que puede ser manipulado visualmente por una herramienta de programación en lenguaje Java.

Entorno web
Componente hecho en software que se puede reutilizar y que puede ser manipulado visualmente por una herramienta de programación en lenguaje Java.

F

Fichero
Bloque lógico de información considerado como una unidad por el usuario. Todos los datos que se almacenan en el ordenador se denominan ficheros. El ordenador puede guardar textos, imágenes, piezas musicales, sonidos y demás. Según su contenido, se almacenan con distintos formatos.

Fichero con valores separados por comas
Fichero usado para transferir datos entre diversos sistemas de gestión, en que los valores de los datos de registro de origen se gravan separando los campos por medio de comas.

G

General packet radio service
Servicio que permite enviar paquetes de datos a través de las redes GSM. Se pueden enviar varios paquetes por distintos canales o aprovechar los "huecos" que se producen en la comunicación y conseguir de esta forma un aprovechamiento más efectivo de los canales de transmisión.

H


HSSI
Interfaz de serie de alta velocidad.

I

Interfaz
Red de redes mundial. Telaraña o entramado mundial. También llamada World Wide Web (WWW), conjunto de redes que permiten la comunicación de millones de usuarios de todo el mundoDispositivo diseñado para facilitar la conexión entre equipos de diferentes funciones o características. f Programa creado para permitir la comunicación entre dos o más aplicaciones diferentes.

Interfaz comun de pasarela
Interfaz que define como se realiza la comunicación entre un servidor web y cualquier otro programa en la misma máquina. Es un programa residente que se ejecuta en el servidor de Internet cuando se accede a una determinada URL y se utiliza para sacar datos del servidor web o introducirlos.

J

Java
Lenguaje de programación creado por Sun Microsystem para proporcionar más velocidad y facilidad de uso a Internet, es independiente de la plataforma utilizada y está disponible para cualquier navegador de la WWW que admita este lenguaje.

K

Kernel
Núcleo básico del sistema operativo, a partir del cual se establecen las distintas capas para su integración con el hardware, para la entrada y salida de datos, etc.

L

HTML
Lenguaje utilizado para la creación de documentos de hipertexto e hipermedia basado en el uso de etiquetas. Con él se definen la posición, la forma y el funcionamiento de las imágenes, los textos e los hipervínculos incluidos en la página. Es el estándar usado en el World Wide Web.

M

Marca paginas
Lenguaje utilizado para la creación de documentos de hipertexto e hipermedia basado en el uso de etiquetas. Con él se definen la posición, la forma y el funcionamiento de las imágenes, los textos e los hipervínculos incluidos en la página. Es el estándar usado en el World Wide Web.
